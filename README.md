# Расцут 
 
Интерактивна веб-апликација за развивање на самодоверба и емоционална писменост кај адолесценти (11–15 год.), инспирирана од наодите на **HBSC** студијата (Health Behaviour in School-aged Children).
 
Наместо класичен квиз, „Расцут“ нуди пет кратки интерактивни игри. Со секоја завршена активност, растението на екранот расте до целосен цут.
 
## Играј
 
Отвори `index.html` во прелистувач. Нема потреба од инсталација, сè работи локално, во прелистувачот.

 
## Технологии
 
Чист HTML5, CSS3 и JavaScript — без рамки (frameworks), без сервер, без база на податоци. Звучните ефекти се генерираат во реално време преку Web Audio API, без надворешни аудио-датотеки.

 
## Приватност
 
Апликацијата не зачувува ништо надвор од активната сесија во прелистувачот — нема collectирање лични податоци, што е особено важно бидејќи целната група се малолетни лица.
