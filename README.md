# Расцут 🌱
 
Интерактивна веб-апликација за развивање на самодоверба и емоционална писменост кај адолесценти (11–15 год.), изработена како дополнителна задача по ПНУВ, инспирирана од наодите на **HBSC** студијата (Health Behaviour in School-aged Children).
 
Наместо класичен квиз, „Расцут“ нуди пет кратки интерактивни игри. Со секоја завршена активност, растението на екранот расте — од семе до целосен цут.
 
## Играј
 
Отвори `index.html` во прелистувач. Нема потреба од инсталација, сервер или регистрација — сè работи локално, во прелистувачот.
 
## Активности
 
| # | Игра | Механика |
|---|------|----------|
| 1 | Препознај го чувството | Квиз со кратки сценарија |
| 2 | Еден ден на училиште | Разгранета приказна низ еден училишен ден |
| 3 | Улови ги добрите мисли | Рефлексна игра — без бои, мора да се чита |
| 4 | Спари ја мислата | Меморија — мисла спарена со здраво преформулирање |
| 5 | Мисли под притисок | „Удри го кртот“, со сѐ покус тајминг |
 
## Технологии
 
Чист HTML5, CSS3 и JavaScript — без рамки (frameworks), без сервер, без база на податоци. Звучните ефекти се генерираат во реално време преку Web Audio API, без надворешни аудио-датотеки.

 
## Приватност
 
Апликацијата не зачувува ништо надвор од активната сесија во прелистувачот — нема collectирање лични податоци, што е особено важно бидејќи целната група се малолетни лица.
